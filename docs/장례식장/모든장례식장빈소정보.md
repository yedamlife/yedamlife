# 시간당 요금 표시 빈소 객실 리스트

전체 1,081곳 장례식장 DB(`funeral_halls`) 에서 `fee.요금_표시`가 **시간당 단가** 형태로 노출되는 빈소(시설임대료) 객실만 추출.

## 분류 기준

- **🔴 정확 매칭**: `임대내용`에서 추출한 시간당 단가(N) ≈ `요금_표시` (N×1).
- **🟠 추정**: 시간당 키워드는 있으나 단가 추출 실패. `요금 == 요금_표시` 이고 금액이 작아(<200,000) 시간당으로 추정.
- **⚪ 불명**: 시간당 키워드 있음. 단가 추출 실패 + 휴리스틱도 일치 안 함 — 수동 확인 필요.

## 객실 업데이트 키

각 행은 `funeral_halls.facility_fees` JSONB 배열의 한 원소. 업데이트 시 식별자:

- `facility_cd` (장례식장)
- `fee_index` (`facility_fees` 배열 내 0-base 인덱스)

예: `update funeral_halls set facility_fees = jsonb_set(facility_fees, '{<fee_index>,구분}', '"hourly"') where facility_cd = '<facility_cd>';`
