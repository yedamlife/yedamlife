- 약식 제사는 정식 제사 대비 약 **51%** (절반 수준)

- 가장 저렴한 기독교 헌화와 가장 비싼 정식 제사의 차이: **475,000원** (약 7.3배)

## 2. 제단 장식

| 유형 | 소형실 | 중형실 | 특실 |

| 일반형 | 700,000원 | 700,000원 | 700,000원 |

| 최고급형 | 2,000,000원 | 2,000,000원 | 2,000,000원 |

### 비교
- 특선형은 일반형 대비 **+300,000원** (약 1.43배)
- 최고급형은 일반형 대비 **+1,300,000원** (약 2.86배), 특선형 대비 **2배**
- 등급 간 가격 간격이 비교적 명확 (700 → 1,000 → 2,000)

## 종합 요약 (제사 + 제단 장식 합산)

| 제사 \ 제단 | 일반형 | 특선형 | 최고급형 |
| --- | ---: | ---: | ---: |
| 정식 제사 | 1,250,000원 | 1,550,000원 | 2,550,000원 |
| 약식 제사 | 980,000원 | 1,280,000원 | 2,280,000원 |
| 기독교 헌화 | 775,000원 | 1,075,000원 | 2,075,000원 |

- **최저가 조합**: 기독교 헌화 + 일반형 = 775,000원
- **최고가 조합**: 정식 제사 + 최고급형 = 2,550,000원
- 최저↔최고 차이: **1,775,000원** (약 3.3배)
