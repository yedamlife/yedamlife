# 운구의전 비용 계산 로직

## 개요

`ReservationModal` (간편 예약 모달)에서 사용자가 선택한 옵션에 따라 자동 계산되는 운구의전 비용 산정 기준입니다.

- **지역 구분**: 수도권(서울·경기·인천) vs 비수도권
- **장법 구분**: 화장 vs 매장
- **운구 인원**: 2명 / 4명 / 6명 / 8명

## 가격표

### 화장

| 지역 | 2명 | 4명 | 6명 | 8명 |
|---|---|---|---|---|
| 수도권 | 240,000원 | 450,000원 | 720,000원 | 960,000원 |
| 비수도권 | 300,000원 | 550,000원 | 900,000원 | 1,200,000원 |

### 매장

| 지역 | 2명 | 4명 | 6명 | 8명 |
|---|---|---|---|---|
| 수도권 | 300,000원 | 600,000원 | 800,000원 | 1,200,000원 |
| 비수도권 | 360,000원 | 720,000원 | 1,000,000원 | 1,440,000원 |

## 가격표 데이터 (코드용)

```ts
const PRICE_TABLE: Record<
  '수도권' | '비수도권',
  Record<'화장' | '매장', Record<2 | 4 | 6 | 8, number>>
> = {
  수도권: {
    화장: { 2: 240_000, 4: 450_000, 6: 720_000, 8: 960_000 },
    매장: { 2: 300_000, 4: 600_000, 6: 800_000, 8: 1_200_000 },
  },
  비수도권: {
    화장: { 2: 300_000, 4: 550_000, 6: 900_000, 8: 1_200_000 },
    매장: { 2: 360_000, 4: 720_000, 6: 1_000_000, 8: 1_440_000 },
  },
};
```

## 지역 판정

수도권으로 분류되는 시/도:

- 서울특별시 (서울)
- 경기도 (경기)
- 인천광역시 (인천)

그 외 시/도는 모두 비수도권으로 처리합니다.

```ts
const METRO_REGIONS = ['서울', '경기', '인천'];
const isMetro = (region: string) =>
  METRO_REGIONS.some((r) => region.includes(r));
```

## 추가 요금

- **의장대 복장 선택**: 인당 +20,000원 (선택 시 인원수 × 20,000원이 합산됨)
- 그 외 추가 요금 없음

## UI 노출 규칙

- 사용자가 지역 / 장법 / 인원을 선택하면 해당 가격을 즉시 표시
- 선택이 누락된 경우 가격 영역은 "옵션을 선택해주세요" placeholder 표시
- 의장대 복장 토글 시 가산 금액을 별도 라인으로 표시하고 합산 가격을 강조 표시
