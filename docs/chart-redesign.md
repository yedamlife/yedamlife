# 장례 비용 비교 차트 리디자인

## 현재 구조

- 3개 막대: 예담라이프 / 선불제상조 / 장례식장
- 각 막대가 **총 비용 1색**으로 표시
- 기준선: 예담라이프 상단에 점선

## 변경 방향

### 핵심 컨셉

**총 비용 = 장례식장 비용(공통) + 상조 비용(차이)**

- 장례식장 비용: 3사 모두 **동일** (장례식장에 내는 비용은 어디를 쓰든 같음)
- 상조 비용: 3사 **각각 다름** (서비스 수수료/가입비 차이)

### 차트 구조

```
        [130만원]
           ▲
           │  상조 비용     상조 비용     상조 비용
           │  (진한색)      (진한색)      (진한색)
           │   30만원       100만원       250만원
           │ ─────────── 가로선: 장례식장 비용 기준 ───────────
           │  장례식장비용   장례식장비용   장례식장비용
           │  (연한색)      (연한색)      (연한색)
           │  100만원       100만원       100만원
           ▼
         예담라이프      선불제상조      장례식장
```

### 데이터 구조 변경

```ts
// 기존
{ yedam: 130, prepaid: 200, funeral: 350 }

// 변경 — 장례식장 비용을 공통으로 분리
{
  facilityFee: 100,      // 장례식장 비용 (3사 공통)
  yedamFee: 30,           // 예담 상조 비용
  prepaidFee: 100,        // 선불제 상조 비용
  funeralHomeFee: 250,    // 장례식장 상조 비용
}
// 총합: yedam=130, prepaid=200, funeral=350 (기존과 동일)
```

### 막대 시각화

각 막대는 **2단 스택 바**:

| 영역 | 위치 | 색상 | 내용 |
|------|------|------|------|
| 상조 비용 | 상단 | 진한색 (예담: 브랜드블루, 타사: gray-400) | 각 업체 서비스 비용 |
| 장례식장 비용 | 하단 | 연한색 (예담: 연블루, 타사: gray-200) | 장례식장 시설 이용료 (공통) |

### 기준선

- 장례식장 비용 상단에 **가로선** 배치
- 라벨: `장례식장 비용 (공통) OO만원`
- 이 선 위로 올라가는 부분이 "상조 비용 = 각 업체별 차이"

### 절약 효과

- 우측 절약 말풍선은 유지
- **상조 비용 기준 절약률** 강조 가능 (장례식장 비용은 어차피 동일하므로)
