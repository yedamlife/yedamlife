## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls` 테이블의 `facility_fees` / `service_items` JSONB 배열을 unnest해서
**판매여부=Y** 이고 **5만원 이상**인 항목만 집계.

### 한 번에 버스/리무진 통계 가져오기

```sql
WITH vehicle_fees AS (
  SELECT
    h.facility_cd,
    (fee->>'요금')::int AS amount,
    fee::text LIKE '%버스%'   AS is_bus,
    fee::text LIKE '%리무진%' AS is_limo
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
  ) AS fee
  WHERE fee->>'판매여부' = 'Y'
    AND (fee->>'요금')::int >= 50000
    AND (fee::text LIKE '%버스%' OR fee::text LIKE '%리무진%')
)
SELECT
  COUNT(DISTINCT facility_cd)::int                                                                  AS hall_count,
  ROUND(AVG(amount) FILTER (WHERE is_bus  AND NOT is_limo))::int                                    AS bus_avg,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount) FILTER (WHERE is_bus  AND NOT is_limo)::int   AS bus_median,
  COUNT(*)        FILTER (WHERE is_bus  AND NOT is_limo)::int                                       AS bus_count,
  ROUND(AVG(amount) FILTER (WHERE is_limo AND NOT is_bus))::int                                     AS limo_avg,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount) FILTER (WHERE is_limo AND NOT is_bus)::int    AS limo_median,
  COUNT(*)        FILTER (WHERE is_limo AND NOT is_bus)::int                                        AS limo_count
FROM vehicle_fees;
```

기대 결과:
```
hall_count | bus_avg | bus_median | bus_count | limo_avg | limo_median | limo_count
-----------+---------+------------+-----------+----------+-------------+-----------
       230 |  466313 |     450000 |       288 |   511908 |      500000 |        174
```

### Supabase RPC 함수로 감싸기

```sql
CREATE OR REPLACE FUNCTION get_vehicle_stats()
RETURNS TABLE(
  hall_count int,
  bus_avg int, bus_median int, bus_count int,
  limo_avg int, limo_median int, limo_count int
)
LANGUAGE sql STABLE AS $$
  WITH vehicle_fees AS (
    SELECT
      h.facility_cd,
      (fee->>'요금')::int AS amount,
      fee::text LIKE '%버스%'   AS is_bus,
      fee::text LIKE '%리무진%' AS is_limo
    FROM funeral_halls h
    CROSS JOIN LATERAL jsonb_array_elements(
      COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
    ) AS fee
    WHERE fee->>'판매여부' = 'Y'
      AND (fee->>'요금')::int >= 50000
      AND (fee::text LIKE '%버스%' OR fee::text LIKE '%리무진%')
  )
  SELECT
    COUNT(DISTINCT facility_cd)::int,
    ROUND(AVG(amount) FILTER (WHERE is_bus  AND NOT is_limo))::int,
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount) FILTER (WHERE is_bus  AND NOT is_limo)::int,
    COUNT(*) FILTER (WHERE is_bus  AND NOT is_limo)::int,
    ROUND(AVG(amount) FILTER (WHERE is_limo AND NOT is_bus))::int,
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount) FILTER (WHERE is_limo AND NOT is_bus)::int,
    COUNT(*) FILTER (WHERE is_limo AND NOT is_bus)::int
  FROM vehicle_fees;
$$;
```

```ts
const { data } = await supabase.rpc('get_vehicle_stats');
const { hall_count, bus_avg, limo_avg } = data[0];
// `가격 정보 제공하는 ${hall_count}개 장례식장`
// `장의버스 평균 ${bus_avg.toLocaleString()}원, 리무진 평균 ${limo_avg.toLocaleString()}원`
```
