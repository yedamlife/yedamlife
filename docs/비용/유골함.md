# 유골함 비용 제공 장례식장

전체 1,081개 장례식장 DB(`funeral_halls`) 스캔 결과.

## 검색 조건

`funeral_supplies` JSONB 배열 항목 중:

- `판매여부 = "Y"` 그리고 `판매구분 = "판매"` (미등록 제외)
- `품종 = "유골함"`
- `요금 >= 10,000` (outlier 제외)
- 장례식장별 가장 낮은 가격 1건 추출

> 재질(`품종상세`)은 `도자기` / `나무` / `나전칠기` / `기타`로 구분되어 있음.

## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls.funeral_supplies` JSONB 배열을 unnest해서 유골함 항목만 집계.

### 통계 쿼리 (장례식장별 최저가 평균)

```sql
WITH urn_items AS (
  SELECT
    h.facility_cd,
    (item->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.funeral_supplies, '[]'::jsonb)
  ) AS item
  WHERE item->>'판매여부' = 'Y'
    AND item->>'판매구분' = '판매'
    AND item->>'품종' = '유골함'
    AND (item->>'요금')::int >= 10000
),
cheapest_per_hall AS (
  SELECT DISTINCT ON (facility_cd) facility_cd, amount
  FROM urn_items
  ORDER BY facility_cd, amount ASC
)
SELECT
  COUNT(*)::int                                              AS hall_count,
  ROUND(AVG(amount))::int                                    AS avg_cheapest,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int   AS median_cheapest,
  MIN(amount)::int                                           AS min_cheapest,
  MAX(amount)::int                                           AS max_cheapest
FROM cheapest_per_hall;
```

기대 결과:
```
hall_count | avg_cheapest | median_cheapest | min_cheapest | max_cheapest
-----------+--------------+-----------------+--------------+-------------
       168 |       71915 |          30000 |        10000 |       500000
```

### 재질별 최저가 평균 쿼리

```sql
WITH urn_by_material AS (
  SELECT
    h.facility_cd,
    item->>'품종상세' AS material,
    (item->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.funeral_supplies, '[]'::jsonb)
  ) AS item
  WHERE item->>'판매여부' = 'Y'
    AND item->>'판매구분' = '판매'
    AND item->>'품종' = '유골함'
    AND (item->>'요금')::int >= 10000
),
cheapest_per_hall AS (
  SELECT DISTINCT ON (facility_cd, material)
    facility_cd, material, amount
  FROM urn_by_material
  ORDER BY facility_cd, material, amount ASC
)
SELECT
  material,
  COUNT(*)::int                                              AS hall_count,
  ROUND(AVG(amount))::int                                    AS avg_cheapest,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int   AS median_cheapest
FROM cheapest_per_hall
GROUP BY material
ORDER BY hall_count DESC;
```

      AND (item->>'요금')::int >= 10000
  ),
  cheapest_per_hall AS (
    SELECT DISTINCT ON (facility_cd) facility_cd, amount
    FROM urn_items
    ORDER BY facility_cd, amount ASC
  )
  SELECT
    COUNT(*)::int,
    ROUND(AVG(amount))::int,

35099;
