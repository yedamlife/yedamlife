# 상복 비용 제공 장례식장

전체 1,081개 장례식장 DB(`funeral_halls`) 스캔 결과.

## 검색 조건

`funeral_supplies` JSONB 배열 항목 중:

- `판매여부 = "Y"` 그리고 `판매구분 = "판매"` (미등록 제외)
- `품종 = "상주용품"`
- `품종상세`가 `남상복` 또는 `여상복` (또는 `품명`에 "상복" 포함)
- `요금 >= 5,000` (outlier 제외)
- 장례식장별 가장 낮은 가격 1건 추출

## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls.funeral_supplies` JSONB 배열을 unnest해서 상복 항목만 집계.

### 통계 쿼리 (장례식장별 최저가 평균)

```sql
WITH mourning_items AS (
  SELECT
    h.facility_cd,
    (item->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.funeral_supplies, '[]'::jsonb)
  ) AS item
  WHERE item->>'판매여부' = 'Y'
    AND item->>'판매구분' = '판매'
    AND item->>'품종' = '상주용품'
    AND (
      item->>'품종상세' IN ('남상복', '여상복')
      OR item->>'품명' LIKE '%상복%'
    )
    AND (item->>'요금')::int >= 5000
),
cheapest_per_hall AS (
  SELECT DISTINCT ON (facility_cd) facility_cd, amount
  FROM mourning_items
  ORDER BY facility_cd, amount ASC
)
SELECT
  COUNT(*)::int                                              AS hall_count,
  ROUND(AVG(amount))::int                                    AS avg_cheapest,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int   AS median_cheapest,
  MIN(amount)::int                                           AS min_cheapest,
  MAX(amount)::int                                           AS max_cheapest
FROM cheapest_per_hall;
```

```

-----------+--------------+-----------------+--------------+-------------
       512 |        19083 |           20000 |         5000 |       60000
```

### 성별 분리 통계 쿼리 (남상복/여상복)

```sql
WITH mourning_by_gender AS (
  SELECT
    h.facility_cd,
    item->>'품종상세' AS gender,
    (item->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.funeral_supplies, '[]'::jsonb)
  ) AS item
  WHERE item->>'판매여부' = 'Y'
    AND item->>'판매구분' = '판매'
    AND item->>'품종' = '상주용품'
    AND item->>'품종상세' IN ('남상복', '여상복')
    AND (item->>'요금')::int >= 5000
),
cheapest_per_hall AS (
  SELECT DISTINCT ON (facility_cd, gender)
    facility_cd, gender, amount
  FROM mourning_by_gender
  ORDER BY facility_cd, gender, amount ASC
)
SELECT
  gender,
  COUNT(*)::int                                              AS hall_count,
  ROUND(AVG(amount))::int                                    AS avg_cheapest,

  MIN(amount)::int                                           AS min_cheapest,
  MAX(amount)::int                                           AS max_cheapest
FROM cheapest_per_hall
GROUP BY gender
ORDER BY gender;
```

### 장례식장별 최저가 리스트 (UI용)

```sql
SELECT DISTINCT ON (h.facility_cd)
  h.facility_cd,
  h.company_name,
  h.full_address,
  item->>'품명'        AS mourning_name,
  item->>'품종상세'    AS gender,
  (item->>'요금')::int AS amount
FROM funeral_halls h
CROSS JOIN LATERAL jsonb_array_elements(
  COALESCE(h.funeral_supplies, '[]'::jsonb)
) AS item
WHERE item->>'판매여부' = 'Y'
  AND item->>'판매구분' = '판매'
  AND item->>'품종' = '상주용품'
  AND (
    item->>'품종상세' IN ('남상복', '여상복')
    OR item->>'품명' LIKE '%상복%'
  )
  AND (item->>'요금')::int >= 5000
ORDER BY h.facility_cd, (item->>'요금')::int ASC;
```
