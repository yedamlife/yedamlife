## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls` 테이블의 `facility_fees` / `service_items` JSONB 배열을 unnest해서
**판매여부=Y** 이고 **요금 > 0** 인 항목만 집계.
정확 키워드(`장례지도사`/`입관지도사`) + 유사 키워드(`염사`/`염습사`/`입관사`/`장례사`/`장의사`/`지도사`/`염습지도`/`인력`/`인건비`)를 모두 포함.

### 통계 쿼리

```sql
WITH director_fees AS (
  SELECT
    h.facility_cd,
    (fee->>'요금')::int AS amount,
    (fee::text LIKE '%장례지도사%' OR fee::text LIKE '%입관지도사%') AS is_exact
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
  ) AS fee
  WHERE fee->>'판매여부' = 'Y'
    AND (fee->>'요금')::int > 0
    AND (
      fee::text LIKE '%장례지도사%' OR fee::text LIKE '%입관지도사%'
      OR fee::text LIKE '%염사%'    OR fee::text LIKE '%염습사%'
      OR fee::text LIKE '%입관사%'  OR fee::text LIKE '%장례사%'
      OR fee::text LIKE '%장의사%'  OR fee::text LIKE '%지도사%'
      OR fee::text LIKE '%염습지도%' OR fee::text LIKE '%인력%'
      OR fee::text LIKE '%인건비%'
    )
)
SELECT
  COUNT(DISTINCT facility_cd)::int                                          AS hall_count,
  COUNT(DISTINCT facility_cd) FILTER (WHERE is_exact)::int                  AS hall_count_exact,
  ROUND(AVG(amount))::int                                                   AS avg_amount,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int                  AS median_amount,
  MIN(amount)::int                                                          AS min_amount,
  MAX(amount)::int                                                          AS max_amount,
  COUNT(*)::int                                                             AS sample_count
FROM director_fees;
```

기대 결과:
```
hall_count | hall_count_exact | avg_amount | median_amount | min_amount | max_amount | sample_count
-----------+------------------+------------+---------------+------------+------------+-------------
        88 |               25 |     271625 |        300000 |      11000 |     700000 |          144
```

### Supabase RPC 함수

```sql
CREATE OR REPLACE FUNCTION get_director_stats()
RETURNS TABLE(
  hall_count int,
  hall_count_exact int,
  avg_amount int,
  median_amount int,
  min_amount int,
  max_amount int,
  sample_count int
)
LANGUAGE sql STABLE AS $$
  WITH director_fees AS (
    SELECT
      h.facility_cd,
      (fee->>'요금')::int AS amount,
      (fee::text LIKE '%장례지도사%' OR fee::text LIKE '%입관지도사%') AS is_exact
    FROM funeral_halls h
    CROSS JOIN LATERAL jsonb_array_elements(
      COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
    ) AS fee
    WHERE fee->>'판매여부' = 'Y'
      AND (fee->>'요금')::int > 0
      AND (
        fee::text LIKE '%장례지도사%' OR fee::text LIKE '%입관지도사%'
        OR fee::text LIKE '%염사%'    OR fee::text LIKE '%염습사%'
        OR fee::text LIKE '%입관사%'  OR fee::text LIKE '%장례사%'
        OR fee::text LIKE '%장의사%'  OR fee::text LIKE '%지도사%'
        OR fee::text LIKE '%염습지도%' OR fee::text LIKE '%인력%'
        OR fee::text LIKE '%인건비%'
      )
  )
  SELECT
    COUNT(DISTINCT facility_cd)::int,
    COUNT(DISTINCT facility_cd) FILTER (WHERE is_exact)::int,
    ROUND(AVG(amount))::int,
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int,
    MIN(amount)::int,
    MAX(amount)::int,
    COUNT(*)::int
  FROM director_fees;
$$;
```
