# 영정사진 비용 제공 장례식장

## 검색 조건

`service_items` JSONB 배열 항목 중:

- `판매여부 = "Y"`
- `품종 = "영정사진"` 그리고 `품종상세 = "사진"` (액자·기타 제외)
- `요금 >= 10,000` (outlier 제외)
- 장례식장별 가장 낮은 가격 1건 추출

> 액자(`품종상세 = "액자"`)는 별도 항목이라 본 집계에서 제외. 영정 액자까지 포함하려면 검색 조건을 풀면 됨.

## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls.service_items` JSONB 배열을 unnest해서 영정사진 항목만 집계.

### 통계 쿼리 (장례식장별 최저가 평균)

```sql
WITH portrait_items AS (
  SELECT
    h.facility_cd,
    (item->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.service_items, '[]'::jsonb)
  ) AS item
  WHERE item->>'판매여부' = 'Y'
    AND item->>'품종' = '영정사진'
    AND item->>'품종상세' = '사진'
    AND (item->>'요금')::int >= 10000
),
cheapest_per_hall AS (
  SELECT DISTINCT ON (facility_cd) facility_cd, amount
  FROM portrait_items
  ORDER BY facility_cd, amount ASC
)
SELECT
  COUNT(*)::int                                              AS hall_count,
  ROUND(AVG(amount))::int                                    AS avg_cheapest,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int   AS median_cheapest,
  MIN(amount)::int                                           AS min_cheapest,
  MAX(amount)::int                                           AS max_cheapest
FROM cheapest_per_hall;
```

기대 결과:
```
hall_count | avg_cheapest | median_cheapest | min_cheapest | max_cheapest
-----------+--------------+-----------------+--------------+-------------
       281 |       82747 |          80000 |        10000 |       800007
```

### Supabase RPC 함수

```sql
CREATE OR REPLACE FUNCTION get_portrait_stats()
RETURNS TABLE(
  hall_count int,
  avg_cheapest int,
  median_cheapest int,
  min_cheapest int,
  max_cheapest int
)
LANGUAGE sql STABLE AS 21511
  WITH portrait_items AS (
    SELECT h.facility_cd, (item->>'요금')::int AS amount
    FROM funeral_halls h
    CROSS JOIN LATERAL jsonb_array_elements(
      COALESCE(h.service_items, '[]'::jsonb)
    ) AS item
    WHERE item->>'판매여부' = 'Y'
      AND item->>'품종' = '영정사진'
      AND item->>'품종상세' = '사진'
      AND (item->>'요금')::int >= 10000
  ),
  cheapest_per_hall AS (
    SELECT DISTINCT ON (facility_cd) facility_cd, amount
    FROM portrait_items
    ORDER BY facility_cd, amount ASC
  )
  SELECT
    COUNT(*)::int,
    ROUND(AVG(amount))::int,
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int,
    MIN(amount)::int,
    MAX(amount)::int
  FROM cheapest_per_hall;
21511;
```

