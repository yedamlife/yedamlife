## 통계 집계 SQL (Postgres / Supabase)

### 전체 통계 쿼리

```sql
WITH anchisil_fees AS (
  SELECT
    h.facility_cd,
    h.sido_cd,
    (fee->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
  ) AS fee
  WHERE fee->>'판매여부' = 'Y'
    AND fee::text LIKE '%안치%'
    AND (fee->>'요금')::int > 0
    AND (fee->>'요금')::int < 300000   -- 30만원 이상 제외
)
SELECT
  COUNT(DISTINCT facility_cd)::int                         AS hall_count,
  ROUND(AVG(amount))::int                                  AS avg_amount,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int AS median_amount,
  MIN(amount)::int                                         AS min_amount,
  MAX(amount)::int                                         AS max_amount,
  COUNT(*)::int                                            AS sample_count
FROM anchisil_fees;
-- 기대: hall_count=844, avg_amount=77926, median_amount=84000
```

## 특이사항

- 안치실이용료는 전국 장례식장의 약 **79%** (853/1,082)가 공시 — 가장 공시율이 높은 항목 중 하나
- **부산·인천·울산·광주**가 평균 10~12만원대로 상대적으로 높음
- **강원** 지역은 중간값 5,000원으로 매우 낮게 집계되는데, 코드성 단가(1,000~5,000원) 등록 비율이 높아 실질 평균과 괴리가 있을 수 있음
- **울산** 전체 16건이 10~15만원 구간에 몰려 있어 표준화 수준이 높은 편
- 항목명 다양: `안치료`, `안치실이용료`, `안치실 냉장고`, `안치비` 등 통일된 명칭 없음
