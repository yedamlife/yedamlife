## 집계 기준 (사이즈 분류)

| 규모 | 면적 범위 | 약 평수 | 기준 |
|---|---|---|---|
| 소형 | `평수_㎡` ≥ 0 AND < 100 | 30평 내외 | 동시 수용 16~24명 |
| 중형 | `평수_㎡` ≥ 100 AND < 198 | 50평 내외 | 동시 수용 40~48명 |
| 대형 | `평수_㎡` ≥ 200 AND < 300 | 80평 내외 | 동시 수용 64~72명 |
| 특실 | `평수_㎡` ≥ 300 AND < 390 | 100평 내외 | 동시 수용 100명 이상 |
| VIP실 | `평수_㎡` ≥ 390 | 130평 이상 | 대규모 인원 수용 |

> 198~200㎡ 구간은 중형/대형 어디에도 속하지 않는 경계 구간 (해당 항목은 집계에서 제외됨)

## 빈소 항목 필터 조건

```
품종 = '시설임대료'
판매여부 = 'Y'
평수_㎡ > 0
AND (
  품종상세 LIKE '%빈소%' OR 품명 LIKE '%빈소%'
  OR 품종상세 LIKE '%분향%' OR 품명 LIKE '%분향%'
  OR (평수_㎡ > 30 AND 품명 ~ '\d+호' AND 품명 NOT LIKE '%객실%' AND 품명 NOT LIKE '%영결%' AND 품명 NOT LIKE '%입관%' AND 품명 NOT LIKE '%안치%')
)
```

## 요금 환산 로직

```
시간당/1시간/시간/회 → 요금 × 48  (24시간 × 2일)
1일/하루/일당/24시간/1박       → 요금 × 2   (2일)
1회/회당                       → 요금 × 1   (그대로)
단위 미기재                    → 요금 × 2   (24시간 기준으로 가정)
```

## 특이사항

- 이상치 원인: 일부 대형병원 장례식장이 **시간당** 요금으로 공시 → × 48 처리로 수천만 원으로 집계됨
- 수도권 중형 최고값 58,752,000원: 시간당 1,224,000원짜리 항목으로 추정
- 장례식장별 최저가 기준 중간값이 실제 이용객 체감가에 가장 근접한 통계
- 수도권 외 소형 데이터(52곳)가 수도권 소형(34곳)보다 많음 → 지방 소규모 병원 장례식장 분포
