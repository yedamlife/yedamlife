# 고인 메이크업 서비스 보유 장례식장

전체 장례식장 1,081곳 DB(`funeral_halls`) 스캔 결과, "메이크업" 키워드를 포함한 비용 항목을 가진 장례식장은 **15곳**.

## 평균 비용

| 구분 | 금액 |
|---|---|
| 단독 메이크업 14건 평균 (패키지 제외) | **127,143원** |
| 단독 메이크업 14건 중간값(median) | **125,000원** |
| 무료(0원)·패키지 제외 13건 평균 | **136,923원** |
| 최저 (유료) | 50,000원 |
| 최고 (단독) | 200,000원 |
| 무료 제공 | 1곳 (갑을녹산) |
| 패키지 포함 | 1곳 (사랑의병원, 특수염습 50만원에 포함) |

> 💡 단독 메이크업 항목 기준 시장가는 **약 10만~20만원** 사이가 일반적이며, 평균/중간값 모두 **약 12.5만원** 수준.

## 특이사항

- 카테고리 분포: `염습/입관` 13건 / `시설임대료` 1건(사랑의병원) / `기타` 1건(계명대)
- 대부분 대학병원·종합병원 부속 장례식장에서 제공
- 사랑의병원(안산)은 단독 메이크업 항목이 아닌 **특수염습 패키지(부검·외인사·메이크업 포함)** 50만원
- 갑을녹산병원은 무료 서비스로 표시되어 있음 (별도 안내 필요)

## 통계 집계 SQL (Postgres / Supabase)

`funeral_halls` 테이블의 `facility_fees` / `service_items` JSONB 배열을 unnest해서
**판매여부=Y** 이고 **요금 > 0** 인 단독 메이크업 항목만 집계 (시설임대료 패키지 제외).

### 통계 쿼리

```sql
WITH makeup_fees AS (
  SELECT
    h.facility_cd,
    (fee->>'요금')::int AS amount
  FROM funeral_halls h
  CROSS JOIN LATERAL jsonb_array_elements(
    COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
  ) AS fee
  WHERE fee->>'판매여부' = 'Y'
    AND fee::text LIKE '%메이크업%'
    AND (fee->>'요금')::int > 0          -- 무료(0원) 제외
    AND fee->>'품종' <> '시설임대료'      -- 패키지 항목 제외
)
SELECT
  COUNT(DISTINCT facility_cd)::int                         AS hall_count,
  ROUND(AVG(amount))::int                                  AS avg_amount,
  PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int AS median_amount,
  MIN(amount)::int                                         AS min_amount,
  MAX(amount)::int                                         AS max_amount,
  COUNT(*)::int                                            AS sample_count
FROM makeup_fees;
```

기대 결과:
```
hall_count | avg_amount | median_amount | min_amount | max_amount | sample_count
-----------+------------+---------------+------------+------------+-------------
        13 |     136923 |        100000 |      50000 |     200000 |          13
```

> 💡 무료(갑을녹산 0원)도 포함하려면 `AND (fee->>'요금')::int > 0` 라인을 제거 → **14곳 / 평균 127,143원** (문서 상단 통계와 일치).

### Supabase RPC 함수

```sql
CREATE OR REPLACE FUNCTION get_makeup_stats()
RETURNS TABLE(
  hall_count int,
  avg_amount int,
  median_amount int,
  min_amount int,
  max_amount int,
  sample_count int
)
LANGUAGE sql STABLE AS $$
  WITH makeup_fees AS (
    SELECT
      h.facility_cd,
      (fee->>'요금')::int AS amount
    FROM funeral_halls h
    CROSS JOIN LATERAL jsonb_array_elements(
      COALESCE(h.facility_fees, '[]'::jsonb) || COALESCE(h.service_items, '[]'::jsonb)
    ) AS fee
    WHERE fee->>'판매여부' = 'Y'
      AND fee::text LIKE '%메이크업%'
      AND (fee->>'요금')::int > 0
      AND fee->>'품종' <> '시설임대료'
  )
  SELECT
    COUNT(DISTINCT facility_cd)::int,
    ROUND(AVG(amount))::int,
    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY amount)::int,
    MIN(amount)::int,
    MAX(amount)::int,
    COUNT(*)::int
  FROM makeup_fees;
$$;
```

```ts
const { data } = await supabase.rpc('get_makeup_stats');
const { hall_count, avg_amount } = data[0];
// `메이크업 비용 제공하는 ${hall_count}개 장례식장 평균 비용 ${avg_amount.toLocaleString()}원`
```
